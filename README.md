# 🛠️ TomSzo Kodi Repository

Saját Kodi addon gyűjtemény – több repository, módosított skin és hasznos modulok egy helyen.

### 📦 Mi található benne?

- **repository.tomszo** – saját repository-m
- Harmadik féltől származó repository-k:
  - repository.bingie
  - repository.elementumorg
  - repository.jurialmunkey
  - repository.movieshark
  - repository.streamshark
  - repository.addons4kodi
- **skin.estuary.modv2** – Módosított Estuary skin (Kodi 21 Omega támogatással)
- **script.module.animated_artwork** – Animált artwork támogatás
- **script.module.pvr.artwork** – PVR artwork modul
- **resource.images.moviegenreicons.estuarymod** – Műfaj ikonok az Estuary-hoz
- **service.autosub** – Automatikus felirat letöltő szolgáltatás

### 🚀 Telepítés Kodi-ban (ajánlott módszer)

1. Kodi-ban menj ide: **Beállítások** → **Fájlkezelő** → **Forrás hozzáadása**
2. Az URL mezőbe írd be pontosan:  
   `https://tomszo.github.io/repository.tomszo`
3. Névnek add meg: `TomSzo Repo`
4. Nyomd meg az **OK** gombot

**A repository telepítése:**
- Menj a **Bővítmények** → **Telepítés ZIP fájlból**
- Válaszd ki a `TomSzo Repo` forrást
- Ott látnod kell a `repository.tomszo-1.0.2.zip` fájlt → telepítsd azt
- Várj 5-10 másodpercet, majd **Telepítés repository-ból** → **TomSzo Repository**  
  és innen telepítheted a többi addont és a skin-t
